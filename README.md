# 🌸 Iris Flower Classification 

This project is a part of my **CodeSoft Data Science Internship – Task 4**.
The objective of this project is to build a Machine Learning model that can classify Iris flowers into different species based on their sepal and petal measurements.

## 📌 Project Overview

The Iris Flower Dataset contains three different species of iris flowers:

* **Setosa**
* **Versicolor**
* **Virginica**
Using Machine Learning algorithms, the model learns patterns from flower measurements and predicts the species accurately.
This is one of the most popular beginner-friendly classification projects in Data Science and Machine Learning.

## 📂 Dataset Information
The dataset includes the following features:
* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### 🎯 Target Variable
* Iris Species
